# 🎬 ИдёмВКино — Система бронирования билетов

Дипломный проект по программе Netology «Профессия Фронтенд-разработчик».

## 🔗 Ссылка на проект

👉 **[Открыть проект на GitHub Pages](https://malikamrs.github.io/Diplom__Film/src/index/index.html)**

## 📋 О проекте

Приложение для онлайн-бронирования билетов в кинотеатр. Включает:

- **Главная страница** — расписание сеансов по залам и датам
- **Страница бронирования** — интерактивная схема зала с выбором мест
- **Страница оплаты** — ввод данных карты и переход к билету
- **Электронный билет** — с QR-кодом и информацией о сеансе
- **Административная панель** — управление залами, фильмами и сеансами

## 🛠️ Стек технологий

- **HTML5** — семантическая разметка
- **CSS3** — стилизация, адаптивная вёрстка
- **JavaScript (ES6+)** — вся бизнес-логика
- **Webpack** — сборка проекта
- **Babel** — транспиляция JS
- **ESLint** — линтинг кода
- **Jest** — модульное тестирование
- **Core-js** — полифиллы
- **html-webpack-plugin** — генерация HTML
- **css-loader / style-loader** — обработка CSS
- **url-loader** — обработка медиафайлов
- **webpack-dev-server** — локальный сервер разработки

## 🚀 Запуск проекта локально

```bash
npm install
npm start
```

Проект откроется по адресу: [http://localhost:8080](http://localhost:8080)

```
src/
```
